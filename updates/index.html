Latest Patch

Domain Move & Readme / Balance Pass
Dec 3, 2025
Live
  • New home: The main game is now hosted at https://snakeesc.github.io. The old URL freshfrogs.github.io/snake now redirects here.
  • Updated the in-game Read Me / How To Play panel to better explain timing for sheds, upgrades, and how buffs stack.
  • Locked in the current pacing:
    • Snake sheds every 5 minutes (faster each shed).
    • Common upgrades available every 1 minute.
    • Epic upgrades available every 3 minutes.
    • Upgrades stack and disappear from the pool when their cap is reached.
  • Tweaked various small UI details to make the game feel better in fullscreen and embedded views (Newgrounds, etc.).
Focus: usability & clarity

Recent Changes

Buff & Orb Balance
Late Nov 2025
Tuning
  • Adjusted orb lifetimes so players have a more reasonable window to chase them without instantly dying to the snake.
  • Added caps to certain buff types to prevent extreme late-game stacking from breaking the difficulty curve.
  • Cleaned up upgrade descriptions to be shorter and easier to skim during runs.
Focus: balance & readability
Multi-Snake Support & Readme Overlay
Late Nov 2025
Feature
  • Refined the logic for spawning additional snakes after the maximum sheds are reached, so extra snakes appear without replacing the original.
  • Ensured the Read Me / How To Play panel properly appears before the first run so new players understand the basics.
  • Fixed issues where certain frogs promoted into the arena were missing their proper role icon in the HUD.
Focus: features & bug fixes
